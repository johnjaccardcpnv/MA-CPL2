# Synopsis
Ce jeu réalisé lors du cours MA-CPL2 est une mine de créativité qui a pour unique but de s'amuser dans un scénario innovant avec notre groupe. Le jeu se compose en 3 parties, le matin, le midi et le soir.

Le jeu se passe au japon (askip) et nous suivons Ren, un gars complètement con, qui va, avec les choix du joueur, vivre de grandes aventures dans cette journée.

## Installation
Lancer le exe dans la version windows (game.zip) pour lancer le jeu, le code source est lancable avec l'application Ren'Py et en placant le jeu dans le répertoire de projets de Ren'Py.

## Comment jouer
Le jeu est jouable entièrement au clavier et souris (plus conseillé de n'utiliser que la souris pour les dialogues), les mini-jeux se jouent uniquement au clavier ou mobile, hormis la fin qui est aussi possible à la manette, même si inconfortable.
